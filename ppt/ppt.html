Background

TowHee - Volunteering events near by!

Built on MEAN stack

Features

  1. MongoDB geospatial query
  2. Google Maps API
  3. Google places

MongoDB

Slides are just HTML elements with a class of slide.


  db.events.createIndex({ location : "2dsphere" });    
  db.events.find({
    location: {
        $near: {
            $geometry: {
                type: "Point",
                coordinates: [
                    -121.88,
                    37.38
                ]
            },
            $maxDistance: 5000
        }
    }
})
